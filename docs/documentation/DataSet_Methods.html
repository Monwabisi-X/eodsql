DataSet Methods

The methods implemented by a returned DataSet object depend on how they are @Selected. There are three types of DataSet

  1. Connected
  2. Disconnected
  3. Updatable (as of 1.1)
Method Connected Disconnected Updatable
add(int, Object) no no no
add(Object) no no yes
addAll(Collection) no no yes
addAll(int, Collection) no no no
clear() no no no
close() yes n/a yes
contains(Object) yes yes yes
containsAll(Collection) yes yes yes
disconnect() yes n/a no
get(int) yes yes yes
indexOf(Object) yes yes yes
isConnected() yes n/a yes
isEmpty() yes yes yes
iterator() yes yes yes
lastIndexOf(Object) yes yes yes
listIterator() yes yes yes
listIterator(int) yes yes yes
remove(int) no no yes
remove(Object) no no no
removeAll(Collection) no no no
retainAll(Collection) no no no
set(int, Object) no no yes
size() yes yes yes
subList(int, int) yes yes yes
toArray() disconnects yes yes
toArray(Object[]) disconnects yes yes

n/a = implemented, but no action will be taken.

disconnects = implemented, but will cause the DataSet to disconnect and cache the ResultSet.
